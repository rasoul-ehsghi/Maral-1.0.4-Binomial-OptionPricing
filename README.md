A professional-grade financial application for pricing options using the Binomial Option Pricing Model. Supports multiple asset classes, dividend yields, and early exercise analysis with interactive visualizations.

---

## âœ¨ Features

* **Multi-Asset Support:** Price options on Stocks, Stock Indices, Futures, and Currency pairs.
* **Flexible Calculations:** Choose between volatility-based or manual up/down factor inputs.
* **Early Exercise Analysis:** Visual indicators for optimal early exercise points in American options.
* **Interactive Visualizations:** Real-time binomial tree diagrams with color-coded nodes.
* **Dividend & Yield Support:** Comprehensive modeling for stocks with dividends and currency rate differentials.
* **Parameter Validation:** Automatic no-arbitrage condition checks and probability validation.
